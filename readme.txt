== Description ==

SD Total Privacy is a password protect wordpress plugin which allows you to password protect all of your wordpress blog including all posts and feeds. It is ideal for wedding / new baby / development / family blogs which you want to be kept private.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
